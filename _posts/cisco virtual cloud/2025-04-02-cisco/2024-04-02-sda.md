# IP 라우팅 정리 - 커뮤니티를 이용한 네트워크 조정

------

BGP의 커뮤니티 속성을 이용하면 다양한 라우팅 정책을 편리하게 적용가능하다.

또한 통신회사(ISP)의 인터넷 망을 구성할 때에는 커뮤니티를 사용하지 않으면 힘들정도로 많이 사용된다.
