# AWS Snowball 개요

------

## **Snowball Edge 디바이스**

**RJ45, SFP+ 및 QSFP+ 동선 및 광 인터페이스를 사용한 10Gbps~100Gbps 링크를 지원하는 고속 네트워크 연결을 제공합니다. 모든 암호화는 디바이스 자체에서 수행되므로 데이터 처리 속도를 빨라지고 데이터 전송 시간은 줄어듭니다**

------

## 빠른 데이터 마이그레이션

모든 암호화는 디바이스 자체에서 수행되므로 데이터 처리 속도는 빨라지고 데이터 마이그레이션 시간은 줄어듭니다

------
